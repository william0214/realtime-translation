# 醫療術語字典（Medical Glossary）

## 概述

本文件列出即時雙向翻譯系統使用的醫療術語對照表，確保翻譯的一致性和準確性。

## 使用原則

1. **強制使用**：Quality Pass 翻譯時必須使用本字典的術語翻譯
2. **優先級**：zh→vi（越南語）為主要目標語言
3. **Fallback**：其他語言若無對應翻譯，則 fallback 到越南語
4. **擴展性**：可根據實際需求新增術語

## 保護單位（Protected Units）

以下單位必須在翻譯時保留原樣：

### 藥物劑量單位

- mg, g, kg
- ml, cc, L
- mcg, μg
- IU, U

### 生命徵象單位

- mmHg（血壓）
- ℃, °C, °F（溫度）
- bpm, 次/分（心跳）
- %（百分比）

### 時間單位

- 天, 週, 月, 年
- 小時, 分鐘, 秒
- 次, 回

### 頻率單位

- 次/天, 次/日
- 次/週
- 次/月

## 使用範例

### 範例 1：生命徵象

**原文（中文）**：
> 您的血壓是 120/80 mmHg，心跳 72 次/分，血氧 98%，體溫 36.5℃。

**翻譯（越南語）**：
> Huyết áp của bạn là 120/80 mmHg, nhịp tim 72 lần/phút, oxy máu 98%, nhiệt độ cơ thể 36.5℃.

**說明**：
- 血壓 → huyết áp（使用 glossary）
- 心跳 → nhịp tim（使用 glossary）
- 血氧 → oxy máu（使用 glossary）
- 體溫 → nhiệt độ cơ thể（使用 glossary）
- 所有數字和單位保留原樣

### 範例 2：症狀詢問

**原文（中文）**：
> 您現在有沒有頭痛或噁心的症狀？

**翻譯（越南語）**：
> Bây giờ bạn có đau đầu hoặc buồn nôn không?

**說明**：
- 沒有 → không（使用 glossary，否定詞）
- 頭痛 → đau đầu（使用 glossary）
- 噁心 → buồn nôn（使用 glossary）

### 範例 3：藥物指示

**原文（中文）**：
> 這是止痛藥，每次 500mg，一天吃 3 次。

**翻譯（越南語）**：
> Đây là thuốc giảm đau, mỗi lần 500mg, uống 3 lần một ngày.

**說明**：
- 止痛藥 → thuốc giảm đau（使用 glossary）
- 500mg → 保留原樣（保護單位）
- 3 次 → 3 lần（使用 glossary）
- 天 → ngày（使用 glossary）

### 範例 4：否定詞保護

**原文（中文）**：
> 您有沒有過敏？沒有的話就可以打針了。

**翻譯（越南語）**：
> Bạn có dị ứng không? Nếu không có thì có thể tiêm.

**說明**：
- 沒有 → không có（使用 glossary，否定詞必須保留）
- 過敏 → dị ứng（使用 glossary）
- 打針 → tiêm（使用 glossary）

## 擴展指南

### 新增術語步驟

1. 在 `shared/glossary.ts` 的 `MEDICAL_GLOSSARY` 陣列中新增條目
2. 指定術語類別（category）
3. 提供至少中文（zh）和越南語（vi）翻譯
4. 其他語言可選（id, fil, en）
5. 加入備註說明（note）

### 新增範例

```typescript
{
  zh: "血糖",
  vi: "đường huyết",
  id: "gula darah",
  en: "blood sugar",
  category: "vital_signs",
  note: "常見縮寫：BG 或 BS"
}
```

## 品質保證

### 術語一致性檢查

- Quality Gate 會檢查翻譯是否使用 glossary 中的術語
- 若偵測到術語但未使用正確翻譯，會觸發重試

### 數字與單位保護

- Quality Gate 會檢查數字是否完整保留
- 若原文有數字但譯文沒有，會觸發重試

### 否定詞保護

- Quality Gate 會檢查否定詞是否正確翻譯
- 若原文有否定詞但譯文疑似缺少，會觸發重試
